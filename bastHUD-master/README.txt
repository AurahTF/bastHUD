Thanks for installing bastHUD.

Version: 1.7.5

Notes:

-To edit the destination of the 'Favorite server button' on the main menu, go to bastHUD/resource and open gamemenu.res. 

In there, replace '[IP SERVER]' with the ip adress of your favorite server, and you're good to go! 

-To apply any of the customizations, copy the desired file and paste it in bastHUD/resource/ui.

-To enable any of the provided crosshairs, go to hudlayout.res and change the visible value to '1' of the section of the correct crosshair.
